# Camera Calibration and Object Measurement

## Prerequisites
Install the required external libraries:
    
    pip install opencv-python numpy


## Usage Instructions
1. The script will first process the calibration images.
2. Once the test image opens, click the 4 corners of the object in this exact order:
   - Top-Left
   - Top-Right
   - Bottom-Right
   - Bottom-Left
3. The real-world dimensions will appear in the terminal.
